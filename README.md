# 3-дневный тренажер к техничке QA в ВК

Сайт разделен на два независимых блока по 3 дня:

1. `Теория + собес`
- День 1: теория тестирования и тест-дизайн.
- День 2: SDLC/Agile, клиент-сервер, API.
- День 3: тестовая документация и репетиция технички.

2. `SQL блок`
- День 1: SQL база (`SELECT/WHERE/ORDER BY/LIMIT`).
- День 2: `JOIN/GROUP BY/HAVING/агрегаты`.
- День 3: финальный лайвкодинг и data quality задачи.

Дополнительно:

- отдельная SQL-песочница для кастомных задач;
- раздельный прогресс по модулям `theory-1..3` и `sql-1..3`;
- расширенные квизы в теории и отдельные SQL-квизы;
- отдельный раздел `QA-квизлеты` с большим банком карточек;
- отдельный режим `Mock собес` (таймер + случайные вопросы + случайные SQL-задачи).

## Локальный запуск

```powershell
python -m http.server 8000
```

Открыть в браузере: `http://localhost:8000`.
