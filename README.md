# Подготовка к техничке QA в ВК

Интерактивный статический сайт для подготовки к техническому интервью:

- теория тестирования;
- техники тест-дизайна;
- SDLC/Agile и процессы;
- клиент-сервер, API и логика проверок;
- тестовая документация;
- SQL-лайвкодинг (песочница + автопроверка задач).

## Запуск локально

```powershell
python -m http.server 8000
```

Открыть: `http://localhost:8000`.
