# 3-дневный тренажер к техничке QA в ВК

Сайт ориентирован на подготовку за короткий срок (3 дня):

- теория тестирования и тест-дизайн;
- SDLC/Agile, роли, сущности;
- клиент-сервер и API-проверки;
- тестовая документация;
- SQL лайвкодинг (база + JOIN/агрегаты + data quality задачи).

## Локальный запуск

```powershell
python -m http.server 8000
```

Открыть в браузере: `http://localhost:8000`.
