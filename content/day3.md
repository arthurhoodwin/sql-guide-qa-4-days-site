# 🎯 ДЕНЬ 3 (Остался 1 день): Финальный прогон под техничку

## План дня (8-10 часов)

1. **Тестовая документация** (1.5-2ч)
- test case vs checklist
- bug report: шаги, expected/actual, окружение, severity/priority
- RTM и контроль покрытия

2. **Полное повторение SQL** (3-4ч)
- быстрый прогон базовых запросов
- JOIN/GROUP BY/HAVING
- data quality задачи (дубликаты, расхождения сумм, NULL)
- N-й максимум, сортировки, лимиты

3. **Mock interview** (2-3ч)
- 20-30 мин теория
- 15-20 мин API/архитектура
- 20-30 мин SQL лайвкодинг
- разбор слабых мест

4. **Анти-паника протокол** (1ч)
- проговаривать ход мысли
- уточнять требования
- не молчать, если не знаешь: давать гипотезу и ограничения

## Формула сильного ответа

1. Короткое определение
2. Где применяю
3. Конкретный пример
4. Риск/компромисс

## Цель к концу дня

- Ты проходишь полный пробный техинтервью-сет
- Есть готовые шаблоны ответов и рабочий темп на SQL
