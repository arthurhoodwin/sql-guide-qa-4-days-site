1. **Теория тестирования** (2-3ч)
- уровни: unit / integration / system / UAT
- виды: functional / non-functional
- smoke / sanity / retest / regression
- severity vs priority

2. **Тест-дизайн** (2ч)
- эквивалентные классы
- boundary value analysis
- decision table
- state transition

3. **SQL база** (3ч)
- SELECT / WHERE / ORDER BY / LIMIT / DISTINCT
- простые агрегаты COUNT/SUM/AVG

4. **Мини-репетиция ответов** (1-2ч)
- ответ по шаблону: определение → применение → пример → риск

## Что отработать вслух

- Разницу между retest и regression
- Когда достаточно checklist вместо test case
- Почему границы чаще всего ломаются
